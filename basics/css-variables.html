CSS variables

  • CSS variables

allows you to store a set of property values that you can reuse within your CSS.

CSS Variables are so new that browser support is an issue. They're not supported in any version of Internet Explorer, and only on very new versions of the other browsers.

  • caniuse.com/#feat=css-variables

var() - To use these variables, you have to use the var function, and then use the name of the variables. All the variables start with the name that begins with two dashes

:root - You can also use the root selector to redefine the values for any of the variables. Now when you do this, make sure that you put your CSS rules after you load the Bootstrap CSS. You should note that using these variables doesn't change the colors or other features of existing Bootstrap classes, since those are hard coded into the CSS, but it serves as a way to create new elements based on those colors, breakpoints and fonts.
