Width and Height

Syntax - (size w, h, mx, mh)-(amount 25, 50, 75, 100)

Sizes can be:

  • w - width
  • w - height
  • w - max-width
  • w - max-height

Amounts can be:

  • 25 - 25%
  • 50 - 50%
  • 75 - 75%
  • 100 - 100%

Height doesn't always work the way you think. But it's the same as how all CSS works.
