Properties

Properties are margin and padding.

Margin

Use the letter m to denote the margin property.

This sets both the left and right margins.

class="mx-auto"

Padding

Use the letter p to denote the padding property.

This sets both the top and bottom padding.

class="py-5"

Sides

Side are top, bottom, left, right, top & bottom, left & right.

Top

Use the letter t to denote the top side.

This sets margin top.

class="mt-2"

Bottom

Use the letter b to denote the bottom side.

This sets padding bottom.

class="pb-2"

Left

Use the letter l to denote the left side.

This sets margin left.

class="ml-2"

Right

Use the letter r to denote the right side.

This sets margin right.

class="mr-2"

Left & Right

Use the letter x to denote both left & right sides.

This sets margin on both sides.

class="mx-2"

Top & Bottom

Use the letter y to denote both top & bottom sides.

This sets padding on both top and bottom.

class="py-2"

Sizes

The sizes range from 0 to 5.

  • 0 removes the value
  • 1 is .25rem
  • 2 is .5rem
  • 3 is 1rem
  • 4 is 1.5rem
  • 5 is 3rem

auto (Horizontal centering)

Use auto to set equal margin sides.

This sets padding on both top and bottom.

class="mx-auto"
