Visibility

These two don't really change the display property itself

invisible - make an element not appear on screen, but it will still take up the amount of room that it would normally take up.

visible - only visible to screen readers

The display property

Syntax - d-(breakpoint)-(type)

Type can be any of these:

  • none
  • inline
  • inline-block
  • block
  • table
  • table-cell
  • flex
  • inline-flex

Make an element disappear using 'none'

<div class="d-sm-none">
