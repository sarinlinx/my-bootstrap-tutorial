1. Install Bootstrap and dependencies with npm

2. Create outline of index.html file:

      
      
          
              AWESOME-GAME ///

3. Create a new directory in your root directory named /scss.

4. Create style.scss file with the following contents.

      $bg-color: blue;

      body {
        background-color: $bg-color;
      }

5. Install gulp and browser-sync and gulp-sass. Save them as a development dependency.

npm install gulp browser-sync gulp-sass --save-dev
npm install --global gulp-cli --save-dev

6. The command above only installed the modules in the /node-modules folder. To use gulp, you must create a gulpfile.js file in the project's root directory.

 ***The following few steps add to this new gulpfile.js***

7. Create variables to reference the packages you just installed: 
      var gulp        = require('gulp');
      var browserSync = require('browser-sync').create();
      var sass        = require('gulp-sass');

      NOTE: gulp, browser-sync, and gulp-sass are referencing the name of the directory in the /node_modules directory.



      8. Gulp is a task runner. So you must create a task.
      First, convert your sass files into CSS.

      // Compile sass into CSS & auto-inject into browsers
      gulp.task('sass', function() {
          //2 locations of scss files
          return gulp.src(['node_modules/bootstrap/scss/bootstrap.scss', 'src/scss/*.scss'])
              //use sass which will convert the scss into CSS
              .pipe(sass())
              //place newly compiled CSS into the /src/css folder
              .pipe(gulp.dest("src/css"))
              //notify browser to reload
              .pipe(browserSync.stream());
      });



      // Move the javascript files into our /src/js folder
      gulp.task('js', function() {
          //reference locations of js files
          return gulp.src(['node_modules/bootstrap/dist/js/bootstrap.min.js', 'node_modules/jquery/dist/jquery.min.js', 'node_modules/popper/dist/js/popper.min.js'])
               //move this into the /src/js folder
              .pipe(gulp.dest("src/js"))
               //notify browser to reload
              .pipe(browserSync.stream());
      });



      // Static Server + watching scss/html files
      //sass will run automatically when server is ran
      gulp.task('serve', ['sass'], function() {

          //initialize server in the src folder
          //local static server running on port 3000
          browserSync.init({
              server: "./src"
          });


          //watch bootstrap file for any changes
          gulp.watch(['node_modules/bootstrap/scss/bootstrap.scss', 'src/scss/*.scss'], ['sass']);
          //anytime it's update the browser will update
          gulp.watch("src/*.html").on('change', browserSync.reload);
      });



      //if you type in gulp in command line, the server will start and js will run to move files from node_modules to /src/js folder
      gulp.task('default', ['js','serve']);
