Default Settings

font-size of 16px

line-height is 1.5.

font-family is "Helvetica Neue", Helvetica, Arial, sans-serif.

In addition, all <p> elements have margin-top: 0 and margin-bottom: 1rem (16px by default).

Reboot.css

There is a special portion at the beginning of the Bootstrap styles that's referred to as Reboot.css. Its job is to normalize styles so that they look similar in different devices and browsers.

Unlike most other normalizing CSS templates, it's a bit opinionated. For example Bootstrap uses Rems, a font measurement system that makes it easier to control both the size of fonts across the entire platform, and within individual components. It's going to make it easier for you to manage your layouts because you can do things across the entire layout, or simply within a specific component a lot easier.

Margins

Bootstrap adds margin only at the bottom of elements.

This means the margin-top property is avoided as much as possible and that you will have to add a little bit of spacing at the top of things if that's what you're looking for. But it does make things easier to update because you know that all of the margins are going to be placed at the bottom of elements.

border-box

Box sizing makes it easier to calculate the width of elements. By default, it's turned to border-box, which means that when you specify the width of an element, say a div being 200 pixels wide, it takes into account the built-in padding.

So if you specify that an element like a div is 200 pixels wide, and then you specify 10 pixel padding, the box is still going to be 200 pixels wide. It's not going to add the extra padding, which is what normal browsers would do.

Native font stacks

This means that the default font in Bootstrap is not set to something like Helvetica. But it tries to use whatever the current platform defines as the default sans-serif font. So for example on current Macs, that would be San Francisco. And that's going to yield a smaller CSS file. And again it makes things look great on different devices, because those fonts have specifically been designed for the device that they're in.

Display Headings

Display headings are used to stand out more than normal headings (larger font-size and lighter font-weight)

There are four classes to choose from. Apply these to the header tag.

  • .display-1
  • .display-2
  • .display-3
  • .display-4
<h1 class="display-1">Display 1</h1>
<h1 class="display-2">Display 2</h1>
<h1 class="display-3">Display 3</h1>
<h1 class="display-4">Display 4</h1>

Element tags

The small element tag

The HTML <small> element is used to create a lighter, secondary text in any heading. For example:

<h1>h1 heading <small>secondary text</small></h1>

Example

Notice how the words 'secondary text' are slightly smaller.

The 'lead' class

Used to make a section of text more prominant.

<p class="lead">Sed venenatis velit gravida egestas laoreet. Donec pretium varius purus, at rhoncus lorem tristique a</p>

Example

Notice the difference in the 2nd paragraph text.

Lead Paragraph: Sed venenatis velit gravida egestas laoreet. Donec pretium varius purus, at rhoncus lorem tristique a

Lists

list-unstyled removes bullets and align list elements to left.

For example:

<ul class="list-unstyled">
  <li>content</li>
</ul>

list-inline on the <ul> puts elements on the same line. You must also put list-inline-item on each <li> item.

For example:

<ul class="list-inline">
  <li class="list-inline-item">content</li>
  <li class="list-inline-item">content</li>
  <li class="list-inline-item">content</li>
</ul>

Blockquotes

blockquote automaticlly styles a blockquoted section.

For example:

<blockquote class="blockquote">
  content
</blockquote>

Example

Notice the difference in the 2nd paragraph text.

Normal text: Sed venenatis velit gravida egestas laoreet.
Donec pretium varius purus, at rhoncus lorem tristique a.
Sed venenatis velit gravida egestas laoreet.


Blockquote: Sed venenatis velit gravida egestas laoreet.
Donec pretium varius purus, at rhoncus lorem tristique a.
Sed venenatis velit gravida egestas laoreet.

Align blockquote text

Use text-right or text-left.

For example:

<blockquote class="blockquote text-right">
  content
</blockquote>

Blockquote footer

blockquote-footer adds a footer with someone's name to the blockquote.

For example:

<blockquote class="blockquote">
  content
  <div class="blockquote-footer">
  John 
  </div>
</blockquote>

Example

Adds the name.

Blockquote: Sed venenatis velit gravida egestas laoreet.
Donec pretium varius purus, at rhoncus lorem tristique a.


  John

Blockquote reverse

blockquote-reverse right-aligns blockquote content.

For example:

<blockquote class="blockquote blockquote-reverse">
  content
</blockquote>
