Filling the homepage

Force the homepage screen to span the full height and width of the device.

To do this, add a height of 100% to the body and html elements.

The HTML

<!-- Create HEADER section. This is the first large section a visitor sees -->
  <section id="header"
      <!-- Add a Bootstrap container -->
      <div class="container">

          <div class="row">
              <div class="col-sm text-center wow fadeIn" data-wow-delay="1s" data-wow-duration="2s">
                  <img id="logo" src="img/logo.png" class="img-fluid">
                  other content...
              </div>
          </div>

      </div>
</section>

The CSS

Wrap the first section in a container with an id of header

In that, create a container with a class of 'container'.

In that create a row and col to place the logo and content.

body, html {
  /* Fills entire browser when loaded */
  height: 100%;
  font-family: 'Alegreya Sans', sans-serif;
}

To the header id, add the following:

  • background-size: cover; - This causes the background image to span the full width and height.
  • height: 100%; - causes background image to span vertically.

  • display: flex
  • align-items: center; - center all content in this first section
#header {
  /* background: url(../img/purple-gradient.png) center center no-repeat; */
  background: url(../img/purple-clouds.jpg) center center no-repeat;
  /*span the entire container*/
  background-size: cover;
  /*force the background image to space the full height*/
  height: 100%;
  /* Enable a flex container for this element */
  display: flex;
  /* Center this section vertifcally and horizontally */
  align-items: center;
}
