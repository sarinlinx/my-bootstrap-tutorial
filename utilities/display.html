Display

Change the display property by adding a class of .d-(breakpoint)-(value). Optional values are:

  • none
  • inline
  • inline-block
  • block
  • table
  • table-cell
  • table-row
  • flex
  • inline-flex

display: none

The first div will hide on large screens and above, but show on smaller screens.

You can also mix and match so certain content only disappears on certain size screens. For example, .d-none .d-md-block would only show on md screens and above.

<div class="d-lg-none bg-light text-dark">Hidden content</div>
<div class="bg-primary text-white">Displayed content</div>

display: inline

.d-inline will convert your elment to an inline element (like a span tag). This means it will only expand to the conent inside, not the full width of the container like it normally would.

<div class="d-lg-inline bg-light text-dark">Inline content on large screens and above</div>
<div class="bg-primary text-white">Normal content</div>

Example

Inline content on large screens and above
Normal content

display: block

.d-block will convert your inline element (like a span tag) to a block element which spans the full width of the container.

<span class="d-lg-block bg-light text-dark">Inline element converted to a block element</span>
<span class="bg-primary text-white">Normal span content</span>

display: inline-block

.d-inline-block does several things:

Compared to display: inline, the major difference is that display: inline-block allows to set a width and height on the element. If you've given the element a special size, display:inline will ignore it, but inline-block will keep the custom size you set.

With display: inline-block, the top and bottom margins/paddings are respected, but with display: inline they are not.

Compared to display: block, the major difference is that display: inline-block does not add a line-break after the element, so the element can sit next to other elements.

The following example shows these features in a block of text.

<div>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vestibulum consequat scelerisque elit sit amet consequat. Aliquam erat volutpat. <span class="d-lg-inline-block text-dark inline-block-height">inline-block element with set height</span> <span class="d-lg-inline-block text-dark inline-block-height">inline-block element with set height</span> gravida nisl sit amet facilisis. Nullam cursus fermentum velit sed laoreet.</div>
