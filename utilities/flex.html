Flex utilties

Flex utilities allow you to manage the layout, alignment, and sizing of your page elements.

d-flex transforms direct children elements into flex items. You can then modify the containers and items further with flex properties.

Flex containers

Add a class of .d-flex to a container to make it a flexbox container. This causes child elements to align horizontally.

Add .d-inline-flex to make it an inline flex container.

You can also use breakpoints such as .d-sm-inline-flex.

d-flex

<div class="d-flex p-2 bg-light text-dark">flexbox container. I take up the full width. My children are side by side.
  <div class="p-2 bg-info">I'm a child</div>
  <div class="p-2 bg-warning">So am I</div>
</div>

Horizontal Direction

Left to right

In the container, add a class of .flex-row to align the items from the left. This is default, so you only need to specify it if you must explicitly set it.

You can also use breakpoints. For example: .flex-lg-row.

<div class="d-flex flex-row bg-light text-dark mb-3">
  <div class="p-2 bd-highlight">Flex item 1</div>
  <div class="p-2 bd-highlight">Flex item 2</div>
  <div class="p-2 bd-highlight">Flex item 3</div>
</div>

Justify content (horizontally)

Use 'justify-content' utilities to change the alignment of flex items.

Breakpoints can be used also. For example: .justify-content-md-center.

Align Fill

Use the .flex-fill class on a series of sibling elements to force them into equal widths while taking up all available horizontal space.

You can also use breakpoints: .flex-md-fill.

Especially useful for equal-width, or justified, navigation.

<div class="d-flex align-items-center text-dark bg-light">
    <div class="p-2 bd-highlight">Flex item 1</div>
    <div class="p-2 bd-highlight">Flex item 2</div>
    <div class="p-2 bd-highlight">Flex item 3</div>
    </div>

Margin right
