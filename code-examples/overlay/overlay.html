Header overlay with centered text

This example takes up the full home screen with an image, overlay, and centered text

HTML

Create section with a unique ID and class of 'fullscreen'.

Create a div section with a class of 'overlay'.

Create a container.

Center content as you like. NOTE: The content is vertically aligned in CSS.

<section id="main-header" class="fullscreen">

    <div class="overlay"></div>

    <div class="container  text-center">
        <div class="row d-flex justify-content-center">
            <div class="col-10">
                <h2 class=" align-center py-5 display-1">Title</h2>
                <p class=" align-center pt-2 display-5">Sub Text</p>
            </div>
        </div>
    </div>

</section>

CSS

asdf

#main-header {
  position: relative;
  background-position: 50% 50%;
  background-repeat: no-repeat;
  background-size: cover;
  background-image: url("header-bg.jpg");
}

.fullscreen {
  display: flex;
  display: -webkit-flex;
  display: -moz-flex;
  display: -ms-flex;
  display: -o-flex;

  /* Center vertically */
  align-items: center;
  /* -webkit-align-items: center; */
  min-height: 100vh;
}

.overlay {
  background-color: #000;
  bottom: 0;
  left: 0;
  opacity: .5;
  position: absolute;
  right: 0;
  top: 0;
  z-index: 0;
}

Changing the overlay color

You can use a gradient color for the overlay.

Just change the background-color property to 'background: linerar-gradient.

background: linear-gradient(to right, #fcffc5 0%, #00c5c7 100%);
