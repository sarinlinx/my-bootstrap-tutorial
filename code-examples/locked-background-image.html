Locked background image on scroll

This effect causese the image to stay in the background while you scroll down the page.

NOTE: You cannot use position with this. This is because 'background-attachment' is ALWAYS relative to the viewport, so setting a position: absolute or position: relative will have no affect.

Instead, you must use: background-position: center;

Add HTML

Add a section with an ID.

Inside, add a div with a class of .dark-overlay.

NOTE: Depending on your section, you may have to move around the padding value in the HTML. Placing it on the dark overlay section seems to work also.

DO NOT add padding in your CSS, otherwise the overlay will be off.

<section id="home-heading" class="p-5">
 <div class="dark-overlay">
   <div class="row">
     <div class="col">
       <div class="container pt-5">
         <h1>Are You Ready To Get Started?</h1>
         <p class="d-none d-md-block">Lorem ipsum dolor, sit amet consectetur adipisicing elit. Est eaque magni sit dolores. Nisi, dolor nam modi perspiciatis
           consequatur soluta.</p>
       </div>
     </div>
   </div>
 </div>
</section>

Add CSS

Add a section with an ID.

Inside, add a div with a class of .dark-overlay.

 #home-heading {
 position: relative; //This MAY be needed so the text displays
 min-height: 200px;
 background: url('../img/lights.jpg');
 background-size: cover; //This is needed to stretch the image across the entire screen
 background-position: center; //this positions the image in the background
 background-attachment: fixed;
 background-repeat: no-repeat;
 text-align: center;
 color: #fff;
}

.dark-overlay {
 position: absolute;
 top: 0;
 left: 0;
 width: 100%;
 height: 100%;
 background: rgba(0, 0, 0, 0.7);
}

Example

The page scrolls while the background image stays fixed

This example uses a small image, so you can see where it ends when you scroll.

You should only use a very large image so the entire page is covered.
