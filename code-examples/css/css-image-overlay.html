CSS background overlay with opacity

This section explains how to add an image to the background of a section.

Next, how to use CSS to create a dark opacity over that background image.

HTML code

Put your HTML in a section with an ID.

Within that, add a div with a class of overlay.

Within that, create your basic Bootstrap sections.

But make sure to add a custom class along side of the Bootstrap 'container' class.

<header id="home-section">
  <!-- Create class for dark overlay of background image -->
  <div class="dark-overlay">

    <!-- Content section -->
    <div container">
        <div class="row">
          <div class="col">

CSS code

Add the background image tot the main section.

Make sure it does not repeat and that it covers the entire page.

Give it a height. The Opacity will need to match this height.

To the .dark-overlay class, position it 'absolute' at the top left.

Add a 100% width and height that matches the original image.

#home-section {
  /* Add image */
  background: url(../img/home.jpg);
  background-repeat: no-repeat;
  background-size: cover;
  background-attachment: fixed;
  min-height: 700px;
}

/* Add dark overlay */
/* In the home section, target the .dark-overlay class */
#home-section .dark-overlay {
  /* Overly starts at top left */
  position: absolute;
  top: 0;
  left: 0;
  /* Cover */
  width: 100%;
  /* height should be same as parent */
  min-height: 700px;

  /* Overlay effect */
  background: rgba(0,0,0,0.7);
}
