HTML code

Put your HTML in a section with an ID.

Within that, add a div with a class of overlay.

Within that, create your basic Bootstrap sections.

But make sure to add a custom class along side of the Bootstrap 'container' class.

Example

Add a section with an ID of .home-section.

Add a div with a class of .dark-overlay.

Add a container with a custom class of .home-inner.

<header id="home-section">
  <!-- Create class for dark overlay of background image -->
  <div class="dark-overlay">

    <!-- Content section -->
    <div class="home-inner container">
        <div class="row">
          <div class="col">
