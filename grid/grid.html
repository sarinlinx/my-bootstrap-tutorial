The Grid overview

Bootstrap uses a 12-column grid system that has breakpoints for the following size devices:

you can group the columns together to create wider columns. The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

It uses Flexbox to create complex layouts with minimal code.

Containers

Containers can be used with or without the grid to align content either to the viewport, or center it around a set of breakpoints.

Responsive fixed width that snap to certain breakpoints, or completely fluid containers which take up 100% of the width of the view port, which is the browser window or the width of your device. Inside containers, you can also place content in rows and columns.

Example

<div class="container">
  code...
</div>

Grid elements have a 15px padding on all sides to work with all backgrounds and elements. You can remove this 15px padding by using no-gutters as a class.

Example

<div class="row no-gutters">

2 types of containers

Regular containers - center content and snap to certain grid points.

Fluid containers - full width of the viewport, which means the width of the device or the browser window.

Rows

These can be used with or without the grid to align content either to the viewport, or center it around a set of breakpoints.

Create a row in order to prepare the content that is going to be placed into columns.

It doesn't need to be a <div>, any tag will work.

Example how to add a row

<div class="container">
  <div class="row">
  code...

  </div>
</div>

Columns

Add the column inside the row. Add a number of columns that add up to 12.

If you don't specify a number value, Bootstrap divides by 12 to align the columns.

It doesn't need to be a <div>, any tag will work.

Example of auto-assigned widths

Since this example does not use numbers, it splits the 3 columns equally.

<section class="container">
  <div class="row">
  <article class="col">
    code for column 1
  </article>

  <article class="col">
    code for column 2
  </article>

  <article class="col">
    code for column 3
  </article>
  </div>
</section>

Example of 3 columns specified

Since this example uses the value of 4 for each column, it specifically creates 3 columns.

<section class="container">
<div class="row">
<div class="col-sm-4">
code for column 1
</div>

<div class="col-sm-4">
code for column 2
</div>

<div class="col-sm-4">
code for column 3
</div>
</div>
</section>

Breakpoints

The Bootstrap 4 grid system has five classes to identify a breakpoint.

  • .col- (auto) extra small (less than 576px)
  • .col-sm- small (greater than or equal to 576px)
  • .col-md- medium (greater than or equal to 768px)
  • .col-lg- large (greater than or equal to 992px)
  • .col-xl- extra large (greater than or equal to 1200px)

The classes above can be combined to create more dynamic and flexible layouts.

Tip: Each class scales up, so if you wish to set the same widths for sm and md, you only need to specify sm.

The grid is extremely flexible. You can create multiple layouts for different breakpoints, reset, offset, nest, and even customize the order of the columns at different breakpoints.

How to specify the size of the column

When you create a column, you can assign it two values:

  • The breakpoint at which it should become responsive
  • How many of 12 column spaces should it occupy

Syntax

<div class="col-sm-4">

Since this breakpoint is 'sm', the column will take up the whole width of the browser until it reaches the 'sm' breakpoint (which is 576px or higher).

Offset columns

This is a way to center columns in a larger space. For example, create a 3 column row.

<section class="container">
  <div class="row">
  <article class="col-sm-4">
  code for column 1
  </article>
  <article class="col-sm-4">
  code for column 2
  </article>
  </section>

The problem with this code is that the 3rd column is missing. So if you view the site, a blank space is shown where the 3rd column should be.

You can use an offset to push the columns to the right, thereby centering them. For example, use an offset of 2 to center the two columns, this way everything adds up to 12.

This is an older way of centering elements. In Bootstrap 4, you should use alignment instead.

<section class="container">
    <div class="row">
    <article class="col-sm-4 offset-sm-2">
    code for column 1
    </article>
    <article class="col-sm-4">
    code for column 2
    </article>
    </section>

Nesting columns

To nest columns, you simply create a new row inside an existing column. Now, this will create a new regular 12-column grid inside that existing column.

It may be a good idea to use the no-gutters class (as mentioned above) with nested columns. This removes the default 15px padding.

Changing the column order

Even though the columns may be coded in a specific order, you can use the order class to re-arranged them.

If you do this, make sure to assign a value to all columns, otherwise the order could get messed up.

<section class="container">
  <div class="row">
  <article class="col-sm-4 offset-sm-2 order-2">
  code for column 1
  </article>

  <article class="col-sm-4 order-1">
  code for column 2
  </article>
  </section>
