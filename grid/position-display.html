Flex Container

2 options when working with the Flex Container.

  • Add additional classes to the container which modify how all internal items work
  • Modify individual items in the container with additional classes

Basic Flex Container

Flex elements are block level. Use the keyword 'inline' to make it an inline element.

Add these classes to the container

Syntax - d-(breakpoint)-(-inline)-flex

Direction classes

Syntax - flex-(breakpoint)-(row or column)-(direction)

direction can be 'row' or 'column'.

The 'direction' is optional. If you use -reverse, it will reverse the order of the elements.

  • flex-sm-row - makes the elements go side by side into rows.
  • flex-sm-column - makes the elements stack on top of one another

Order

Control the order of the elements within the flexbox container.

Syntax - order-(breakpoint)-(# 1 through 12)

Example: order-sm-8

Justify

Controls the spacing between items

Syntax - justify-content-(breakpoint)-(alignment-option)

  • justify-content-sm-start - puts items to left and extra space to right
  • justify-content-sm-end - puts items to right and extra space on left
  • justify-content-sm-center - centers horizontally
  • justify-content-sm-around - put the space around and in between elements
  • justify-content-sm-between - does something similar but it makes the edge element align to the edge

Wrap

control whether the elements wrap in relation to the space in their container

Syntax - flex-(breakpoint)-(wrap or nowrap)-(direction)

Direction is optional

  • flex-sm-wrap
  • flex-sm-nowrap

Individual Flex Elements

Align Self

Just like we can control a group of elements in Flex by controlling the container, we can also control individual Flex elements. And the way that that works is by using a class called align-self

Syntax - align-self-(breakpoint)-(align-option)

  • align-self-sm-start
  • align-self-sm-end
  • align-self-sm-center
  • align-self-sm-baseline
  • align-self-sm-stretch

Controlling order of elements in Flexbox

Syntax - order-(breakpoint)-(# 1 through 12)

Example: order-sm-9
