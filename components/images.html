Basic image classes

  • .img-fluid - This creates responsive images.
  • .rounded - Adds a rounded edge to an image. You can also specify which side of the image should be rounded.
    • .rounded-top
    • .rounded-right
    • .rounded-bottom
    • .rounded-left
    • .rounded-circle - make an oval shaped image. If the image happens to be square, then it will actually give you a circle
    • .rounded-rounded-0 - if the image already has some roundness applied to it, you can specify a class of rounded-0 and it's going to take out the roundness of that image if it has round edges
  • .img-thumbnail - a slightly rounded image with also a one pixel border around the image with a little bit of space.

Aligning images

  • .float-left or .float-right - By default, images are considered to be inline elements, so you can float them.
  • .text-center - centers the image and text
  • .mx-auto - if you define that an image needs to be a block element, then you can use the mx-auto class, and it will center the block, which in this case is the images

Figures

  • .figure on the <figure> tag
  • .figure-img on the images
  • .figure-caption on the text
