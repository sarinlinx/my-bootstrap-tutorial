Responsive images

Make an image response by adding the class .img-fluid.

This automatically applies 'max-width: 100%' and 'height: auto' so it scales with the parent.

When you shrink the viewport, the image shrinks with it.

NOTE: SVG images and IE 10. In Internet Explorer 10, SVG images with .img-fluid are disproportionately sized. To fix this, add width: 100% \9; where necessary. This fix improperly sizes other image formats, so Bootstrap doesnâ€™t apply it automatically. 

<img src=".." class="img-fluid" alt="Responsive image">

Thumbnails

Add a class of .img-thumbnail to give it a rounded 1px border.

The image is also responsive.

<img src=".." class="img-thumbnail" alt="Responsive image">

Aligning images

There are three ways to align images:

  • Using a helper float class
  • Using a text alignment class
  • block-level images can use the .mx-auto utility class

Floating images

Using .float-left or .float-right to float an image.

The image sizes must already be small enough to fit in the section. They will not resize to fit the section automatically.

<img src=".." class="float-left">
<img src=".." class="float-right">

Example

Using text-alignment

Put the image in a container with a class of .text-center.

<div class="text-center">
  <img src="..">
</div>

Example

Center block level images

Add both the class of .mx-auto and .d-block (display-block) to center an image.

<img src=".." class="mx-auto d-block">

Figures

Wrap the image in a container with a class of <figure>. Give it a class of .figure to style.

Give the image tag classes of .figure-img to style.

Under the image, add a <figcaption> tag to add a caption to an image. Give it a class of .figure-caption to style.

You can align the text using .text-right on the figcaption tag.

<figure class="figure">
  <img src=".." class="figure-img img-fluid">
  <figcaption class="figure-caption">A caption for the above image.</figcaption>
</figure>

Basic image classes

  • .img-fluid - This creates responsive images.
  • .rounded - Adds a rounded edge to an image. You can also specify which side of the image should be rounded.
    • .rounded-top
    • .rounded-right
    • .rounded-bottom
    • .rounded-left
    • .rounded-circle - make an oval shaped image. If the image happens to be square, then it will actually give you a circle
    • .rounded-rounded-0 - if the image already has some roundness applied to it, you can specify a class of rounded-0 and it's going to take out the roundness of that image if it has round edges
  • .img-thumbnail - a slightly rounded image with also a one pixel border around the image with a little bit of space.

Aligning images

  • .float-left or .float-right - By default, images are considered to be inline elements, so you can float them.
  • .text-center - centers the image and text
  • .mx-auto - if you define that an image needs to be a block element, then you can use the mx-auto class, and it will center the block, which in this case is the images

Figures

  • .figure on the <figure> tag
  • .figure-img on the images
  • .figure-caption on the text
