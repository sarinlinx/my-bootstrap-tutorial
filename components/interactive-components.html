Tooltips

  • Tooltips

you should only use tooltips on HTML elements that are focusable like links and foreign controls because of problems with usability

.data-toggle="tooltip" - In order to set these up, you add a data-toggle of tooltip to an element

title="text" - Then, then add a title attribute with some text in it. That text can optionally have some HTML

Configured in 2 ways

data attribute (easier) or JS configuration

Tooltips are not turned on by default so if you do want to use 'em, you will still need to use JavaScript, even if you're using data attributes.

Example of JS activation - You would create sort of a regular jQuery call, which is what this is, and then you target the elements in the DOM that have a data-toggle of tooltip, or you can call an individual tooltip here if you want to, and then you use the tooltip method and pass along some options. 

$(function () {
  $('[data-toggle="tooltip"]').tooltip({ OPTIONS })
})

Options

.placement-(position) - Where it will show up. The position can be top, right, bottom, left

.trigger - how tooltip is shown. By default this is when you hover, but you can change it to 'click, focus, or hover'.

.html (true or false) - You can also add simple HTML to a tooltip. Remember these will go in the title attribute, so you can put HTML, but if you do, you need to set the data-html attribute to true, because otherwise it'll literally type in the text with the HTML tags

Popovers

  • Popovers

Very similar to tooltips, with a different style.

data-toggle="popover" - add to an element

title="text" - add title attrubte with some text

data-content="content" - add additional content

Activating popovers

You must use jQuery.

$(function () {
  $('[data-toggle="popover"]').popover({ OPTIONS })
})

Options

.placement-(position) - top, right, bottom, left

trigger - (click, hover, focus)

Collapse accordions

Collapse and accordions are a set of related components that allow you to show or hide HTML content. Collapse is the simplest, and lets you show or hide content on a click.

Accordions are similar but part of a bigger group. When one item shows, another hides, so accordions are essentially group of collapses

Collapse

To set up a collapse, you create a link or a button that activates the collapse element.

data-toggle="collapse - Add to that element

#id or data-target- to target the element,use either an ID if it's an anchor tag or link, or you can use a data-target attribute if it's a button.

.collapse - The element that's going to show or hide has the collapse class.
