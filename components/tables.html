Tables

Table classes

Add .table class to the table tag.

.table-dark makes the table dark

.table-striped

.table-bordered

.table-hover

Colors

In head section add .thead-light or .thead-dark.

.table-(colors) on rows or data tags

these table contextual classes are designed to be backgrounds on existing rows. So they're generally lighter.

  • table-active
  • table-primary
  • table-secondary
  • table-success
  • table-danger
  • table-warning
  • table-info
  • table-light
  • table-dark

For darker colors, use any contextual color class

  • bg-primary
  • bg-success
  • bg-danger
  • bg-warning
  • bg-info

Size

Use .table-sm - it's going to make a table that has a little less padding to it. So it'll look a little smaller

.table-responsive-(breakpoint) - so that the table cells scroll at a certain break point.
