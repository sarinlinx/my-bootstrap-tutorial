Overview

Carousel's rely on JavaScript. Make sure you've linked the following:

  • jQuery
  • Popper.js
  • bootstrap.js

The bootstrap.bundle.js file already contains popper.js.

Images

If you're using images, make sure they are already the same size, otherwise the slider will jump around.

Adding a carousel with inline images

Basic layout

Give the opening div an id, class of 'carousel', 'slide' and data=ride="carousel".

Give the 2nd div a class of carousel-inner.

From here, put each image in their own div.

Add a class of carousel-item to each img div

Set the first one to be 'active'.

Add classes to images of d-block and img-fluid so they retain their correct size relative to its container.

<!-- slider -->
<div id="slider1" class="carousel slide mb-5" data-ride="carousel">
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img class="d-block img-fluid" src="#" alt="first slide">
    </div>
    <div class="carousel-item">
      <img  class="d-block img-fluid" src="#" alt="second slide">
    </div>
    <div class="carousel-item">
      <img  class="d-block img-fluid" src="#" alt="third slide">
    </div>
  </div>
</div>

Adding controls

Create left and right arrow icons using anchor tags.

Make sure these are added inside of the main carousel div, but outside the div with a class of 'carousel-inner'.

For the left arrow, give it a class of 'carousel-control-prev'.

For the right arrow, give it a class of 'carousel-control-next'.

Give both the href of the ID of the carousel.

Give both a data-slide rule of 'prev' or 'next'.

Give each an icon in a span tag with 'carousel-control-prev-icon' or 'carousel-control-next-icon'. 

<!-- slider -->
<div id="slider1" class="carousel slide mb-5" data-ride="carousel">
  <div class="carousel-inner">>
    <div class="carousel-item active">
      <img class="d-block img-fluid" src="#" alt="first slide">
    </div>
    <div class="carousel-item">
      <img class="d-block img-fluid" src="#" alt="second slide">
    </div>
    <div class="carousel-item">
      <img class="d-block img-fluid" src="#" alt="third slide">
    </div>
  </div>

  <!-- control arrows -->
  <a class="carousel-control-prev" href="#slider1" data-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </a>

  <a class="carousel-control-next" href="#slider1" data-slide="next">
    <span class="carousel-control-next-icon"></span>
  </a>

</div>

Adding indicators

Add an ordered list just under the first div tag.

Give the first list item a class of active.

Give all inidcators a data-target value of the carousel's ID.

Give each a data-slide-to value to indicate where they should slide.

<!-- slider -->
          <div id="slider1" class="carousel slide mb-5" data-ride="carousel">

<ol class="carousel-indicators">
  <li class="active" data-target="#slider1" data-slide-to="0"></li>
<li data-target="#slider1" data-slide-to="1"></li>
<li data-target="#slider1" data-slide-to="2"></li>
</ol>

Adding captions to the images

Under the image, but within the same block, add another div.

Give it a class of carousel-caption.

This example also sets the display property so the caption only displays on md size screens or larger.

Add your text.

<!-- slider -->
<div id="slider1" class="carousel slide mb-5" data-ride="carousel">

  <ol class="carousel-indicators">
  <li class="active" data-target="#slider1" data-slide-to="0"></li>
  <li data-target="#slider1" data-slide-to="1"></li>
  <li data-target="#slider1" data-slide-to="2"></li>
  </ol>

  <div class="carousel-inner">>

Adding a carousel with background images

This creates an image carousel using CSS background images.

In this way, you do not have to add the images to the HTML.

Everything is the same as above, except for the .carousel-inner section.

Basic layout

For each image div, create a custom class of .carousel-image-#.

Add a .container within the each image div.

<div class="carousel-inner">
    <div class="carousel-item carousel-image-1 active">
        <div class="container">
            <div class="carousel-caption d-none d-sm-block text-right mb-5">
                <h1 class="display-3">Heading One</h1>
                <p class="lead">Lorem ipsum dolor sit amet consectetu.</p>
                <a href="#" class="btn btn-danger btn-lg">Sign Up Now</a>
            </div>
        </div>
    </div>

    <div class="carousel-item carousel-image-2">
        <div class="container">
            <div class="carousel-caption d-none d-sm-block mb-5">
                <h1 class="display-3">Heading Two</h1>
                <p class="lead">Lorem ipsum dolor sit amet consectetur a.</p>
                <a href="#" class="btn btn-primary btn-lg">Learn More</a>
            </div>
        </div>
    </div>

    <div class="carousel-item carousel-image-3">
        <div class="container">
            <div class="carousel-caption d-none d-sm-block text-right mb-5">
                <h1 class="display-3">Heading Three</h1>
                <p class="lead">Lorem ipsum dolor sit amet consectetur .</p>
                <a href="#" class="btn btn-success btn-lg">Learn More</a>
            </div>
        </div>
    </div>
</div>

CSS layout

Give the items a height so you can see them.

background-size: cover makes the image span the entire width.

This will stetch the image, so make sure they are all the same size first.

.carousel-item {
  height: 450px;
}

.carousel-image-1 {
  background: url('../img/carousel-image1.jpg');
  background-size: cover;
}

.carousel-image-2 {
  background: url('../img/carousel-image2.jpg');
  background-size: cover;
}

.carousel-image-3 {
  background: url('../img/carousel-image3.jpg');
  background-size: cover;
}

Center image within carousel

To center an image within the carousel, add the following CSS.

.carousel img {
  margin: auto;
}

JavaScript

This changes how fast the slider moves in milliseconds.

Hover stops the slider when you hover your mouse over it.

//retrieve carousel class
$('.carousel').carousel({
  //Change slide interval in ms
  interval: 3000,
  pause: 'hover'
})
