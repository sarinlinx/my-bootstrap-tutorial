Breadcrumbs

Add a class of .breadcrumb to the <ol> list item.

Add a class of .breadcrumb-item to each list item.

For the current page, use a class of .active.

<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li  class="breadcrumb-item"><a href="#">Home</a></li>
    <li  class="breadcrumb-item active" aria-current="page">Library</li>
  </ol>
</nav>

Accessibility

Since breadcrumbs provide a navigation, itâ€™s a good idea to add meaningful labels.

For the <nav> tag, add aria-label="breadcrumb" to describe the type of navigation provided.

For the active page, add aria-current="page" to indicate that it represents the current page.
