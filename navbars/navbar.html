Navbar

The Navbar is the navigation bar at the top of your page.

To create, add a class of .navbar with .navbar-expand-(breakpoint) for responsive collapsing and color scheme classes.

Navbars and their contents are fluid by default. Use optional containers to limit their horizontal width.

Use our spacing and flex utility classes for controlling spacing and alignment within navbars.

Navbars are hidden by default when printing. Force them to be printed by adding .d-print to the .navbar.

Ensure accessibility by using a <nav> element or, if using a more generic element such as a <div>, add a role="navigation" to every navbar to explicitly identify it as a landmark region for users of assistive technologies.

UL example

Add .navbar to the container for general styling.

Add .navbar-expand-(breakpoint) so the navbar displays normally on certain devices, but as an icon dropdown on smaller devices.

To create the toggle dropdown (hamburger icon), create a <button> with a class of .navbar-toggler. In it, add a span class of .navbar-toggler-icon.

Wrap the nav items in a div with a class of .collapse. This causes the items to hide on smaller devices so only the toggler icon shows.

Also add .navbar-collapse to the item container for styling.

To the unordered list tag, add .navbar-nav for styling. This makes the links horizontal.

<nav class="navbar navbar-expand-lg navbar-light bg-light">

    <!-- Hamburger icon on small devices -->
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
<span class="navbar-toggler-icon"></span>
</button>

    <div class="collapse navbar-collapse" id="navbarSupportedContent">
        <ul class="navbar-nav mr-auto">
            <li class="nav-item active">
                <a class="nav-link" href="#">Home <span class="sr-only">(current)</span></a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Link</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Link</a>
            </li>
        </ul>
    </div>
</nav>

Brand title

Add .navbar-brand to a link under the opening container tag to an anchor tag. This creates a title.

You can add this class to different elements, but adding to an anchor tag is easiest as it doesn't involve more customization.

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar Title</a>

<!-- Hamburger icon on small devices -->
<button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
<span class="navbar-toggler-icon"></span>

Responsive behavior

Navbars can utilize the following classes to change when their content collapses behind a (hamburger icon) button.

  • .navbar-toggler
  • .navbar-collapse
  • .navbar-expand{-sm|-md|-lg|-xl}

For navbars that never collapse, add the .navbar-expand class on the navbar.

For navbars that always collapse, don’t add any .navbar-expand class.

Toggler

To create the toggle dropdown (hamburger icon), create a <button> with a class of .navbar-toggler. In it, add a span class of .navbar-toggler-icon.

By default, navbar togglers are left-aligned.

They are automatically aligned to the far right if they follow a sibling element like a .navbar-brand. Reversing your markup will reverse the placement of the toggler. Below are examples of different toggle styles.

Left aligned dropdown

This example aligns the dropdown icon to the left.

Wrap the nav items in a div with a class of .collapse. This causes the items to hide on smaller devices so only the toggler icon shows.

Also add .navbar-collapse to the item container for styling.

To the unordered list tag, add .navbar-nav for styling. This makes the links horizontal.

<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarTogglerDemo01" aria-controls="navbarTogglerDemo01" aria-expanded="false" aria-label="Toggle navigation">
<span class="navbar-toggler-icon"></span>
</button>
    <div class="collapse navbar-collapse" id="navbarTogglerDemo01">
        <a class="navbar-brand" href="#">Hidden brand</a>
        <ul class="navbar-nav mr-auto mt-2 mt-lg-0">
            <li class="nav-item active">
                <a class="nav-link" href="#">Home <span class="sr-only">(current)</span></a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Link</a>
            </li>
        </ul>
    </div>
</nav>

Trigger hidden content from the dropdown icon

You can make the toggle button trigger hidden content elsewhere on the page.

This is done by matching the 'id' on the collapseable content container, and the 'data-target' on the button.

<div class="pos-f-t">

<div class="collapse" id="navbarToggleExternalContent">
    <div class="bg-dark p-4">
      <h4 class="text-white">Collapsed content</h4>
      <span class="text-muted">Toggleable via the navbar brand.</span>
    </div>
  </div>

  <nav class="navbar navbar-dark bg-dark">
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarToggleExternalContent" aria-controls="navbarToggleExternalContent" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
  </nav>
</div>
