Scrollspy

Navbar

Scrollspy updates your navbar as you scroll down the page.

UL example

Create a navbar and assign it a specific id attribute.

Most often you'll be spying on the 'body' element. To make this work you'll need to do the following to this element.

  • In CSS, make the position of the body relative.
  • To the body tag, add data-spy="scroll".
  • To the body tag, add data-target="#id-of-navbar".
  • It also makes sense to add .sticky-top to the navbar.

As stated above, the element you're spying on is usually the body tag. However, if you want a small navbar to a specific section of the page, just wrap the content of that section in a container.

Give it an id or class to assign it a position of relative. Then add the data-spy and data-target as shown above to the container.

<body data-spy="scroll" data-target="#navbar-example2">

  <nav id="navbar-example2" class="navbar navbar-light bg-light sticky-top">
      <ul class="nav nav-pills">
          <li class="nav-item">
              <a class="nav-link" href="#one">@one</a>
          </li>
          <li class="nav-item">
              <a class="nav-link" href="#two">@two</a>
          </li>
          <li class="nav-item">
              <a class="nav-link" href="#three">@three</a>
          </li>
          <li class="nav-item">
              <a class="nav-link" href="#four">@four</a>
          </li>
      </ul>
  </nav>

<!-- content section -->
  <h4 id="one">one</h4>
    <p>data here</p>
  <h4 id="two">two</h4>
    <p>data here</p>
  <h4 id="three">three</h4>
    <p>data here</p>

Smooth scrolling to the section with JS

Create a navbar and assign it a specific id.

Full code can be found here:

  • Smooth scrolling
// Add scrollspy to body
$('body').scrollspy({target: "#navbar-example3", offset: 50});

// Add smooth scrolling on all links inside the navbar
$("#navbar-example3 a").on('click', function(event) {

  // Make sure this.hash has a value before overriding default behavior
  if (this.hash !== "") {

    // Prevent default anchor click behavior
    event.preventDefault();

    // Store hash
    var hash = this.hash;

    // Using jQuery's animate() method to add smooth page scroll
    // The optional number (800) specifies the number of milliseconds it takes to scroll to the specified area
    $('html, body').animate({
      scrollTop: $(hash).offset().top
    }, 800, function(){

    // Add hash (#) to URL when done scrolling (default click behavior)
      window.location.hash = hash;
    });

  } // End if

});
