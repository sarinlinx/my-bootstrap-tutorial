Basic Form classes

Navs

You can create using an unordered list (more code required), or a div.

Both options use the .nav class as the primary formatting class.

Unordered list option

Add the .nav class to the unordered list tag.

Give each item in the list a class of .nav-item.

Give each link a class of .nav-link.

<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link  href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link  href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link  disabled" href="#">Disabled</a>
  </li>
</ul>

Example

  • Active
  • Link
  • Link
  • Disabled

container tag option

Add the .nav class to the div container.

Give each link a class of .nav-link.

<nav class="nav">
  <a class="nav-link active" href="#">Active</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link disabled" href="#">Disabled</a>

Aligning horizontally

By default, navigation items are left aligned.

To center them, use .justify-content-center on the container.

To right align, use .justify-content-end.

<nav class="nav justify-content-center">
  <a class="nav-link active" href="#">Active</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link disabled" href="#">Disabled</a>
</nav>

Example

Active Link Link Disabled

Aligning vertically

Use .flex-column on the container to stack vertically.

If you wish to specify the size screen, use a breakpoint. For example: flex-sm-column

<nav class="nav flex-column">
  <a class="nav-link active" href="#">Active</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link" href="#">Link</a>
  <a class="nav-link disabled" href="#">Disabled</a>

Fill and Justify content

Fill

Use .nav-fill on the container to fill the full width.

Notice that all horizontal space is occupied, but not every nav item has the same width.

For this to work, you must add .nav-item on the links.

<nav class="nav nav-pills nav-fill">
  <a class="nav-item nav-link active" href="#">Active</a>
  <a class="nav-item nav-link" href="#">Lindfdfddfdfdfd dfdd dfd f fd dfdfk zzzzzzzzzzzz s</a>
  <a class="nav-item nav-link" href="#">Link</a>
  <a class="nav-item nav-link disabled" href="#">Disabled</a>
</nav>

Example

Active Lindfdfddfdfdfd dfdd dfd f fd dfdfk zzzzzzzzzzzz s Link Disabled

Justify

For equal-width elements, use .nav-justified. All horizontal space will be occupied by nav links, but unlike the .nav-fill above, every nav item will be the same width.

Use .nav-justified on the container.

<nav class="nav nav-pills nav-justified">
        <a class="nav-item nav-link active" href="#">Active</a>
        <a class="nav-item nav-link" href="#">Lindfdfddfdfdfd dfdd dfd f fd dfdfk zzzzzzzzzzzz s</a>
        <a class="nav-item nav-link" href="#">Link</a>
        <a class="nav-item nav-link disabled" href="#">Disabled</a>
        </nav>

Working with flex utilities

Flexbox utilities allow you to create responsive navs.

In the example below, our nav will be stacked on the lowest breakpoint, then adapt to a horizontal layout that fills the available width starting from the small breakpoint.

.flex-column sets the nav to vertical, but .flex-sm-row forces it to be horizontal on a sm or larger layout.

<nav class="nav nav-pills flex-column flex-sm-row">
 <a class="flex-sm-fill text-sm-center nav-link active" href="#">Active</a> <a class="flex-sm-fill text-sm-center nav-link" href="#">Link</a> <a class="flex-sm-fill text-sm-center nav-link" href="#">Link</a> <a class="flex-sm-fill
text-sm-center nav-link disabled" href="#">Disabled</a> </nav>

Nav Bars

Nav link options

.active

.disabled - prevent people from clickin' on the link and it sort of shows as a grayed-out link.

Nav Styles

.nav-pills - When you use pills, you will get, in the active link, something that looks like a little button with rounded edges,

.nav-tabs - when you tabs, of course, each one of the elements looks like tabs.

Nav Alignment

.justify-content-center - center a nav

.justify-content-end - right align

.nav-fill - make the links fit the horizontal space, but the space of each link is going to be different depending on the content and the width of the text.

.nav-justified - force the space in between each element to be the same no matter the length of the text.

.flex-column - make your navigation more responsive.

Nav Bars

.navbar class goes on main container.

By default, navbar components stack, so you need to add a navbar-expand class to control when the navbar is going to expand.

<nav class="navbar navbar-expand-lg fixed-top">

.navbar-expand

by default, all the navigation elements will be right on top of one another, and we have to specify when you want them to go horizontal.

Syntax - navbar-expan-(breakpoint)

Navbar Colors

Use .bg-COLOR class to specify Navbar color.

Syntax - bg-(color).

example: bg-primary, bg-danger

Light or Dark

Two classes to specify if the link background will be light or dark.

navbar-light

navbar-dark

navbar-nav

inside this main navbar class, we need to use another container for a navbar-nav, and this is where your main navigation will go.

Inside this container you'll need to use .nav-item and .nav-link.

Specify one element is active using .active

Disable a link using .disabled

Navbar Brand and Text

There are other components you can add to your navigation.

The .navbar-brand is used for text or logos and will get an enhanced look. usually a link or a headline and it gets a special look in Bootstrap

Example: This links a logo

<a class="navbar-brand" href="#"><img src="img/logo-small.png" class="img-fluid"></a>

.navbar-text - separate text added to nav

Both go inside the .navbar-nav component.

Nav Dropdown

Add a container with a class of .dropdown. this container is used to align the dropdown to the proper item in the navigation.

.dropdown-toggle is added on the link in order to get a little triangle to indicate that this element has a dropdown in it.

Dropdowns are activated in 2 ways

Using JavaScript

Using a data attribute. In Bootsrap this would be data-toggle="dropdown"

Building the menu

.dropdown-menu creates the dropdown. Put this in a different container.

Each dropdown link gets .dropdown-item class.

Should also add an id & aria attributes - to make everything work well for accessibility devices.

Adding a form element to a nav bar

Add a container with a class of .form-inline.

Then add .form-control classes as needed.
