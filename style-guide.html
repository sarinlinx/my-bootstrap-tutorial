Live examples of code

A live example of code shows how it actually looks and functions in a browser.

Add a header titled 'Example' in an <h4> tag.

Add the code in a new row with a class of .ex-highlight.

<section class="container">
  <h4>Example</h4>
    <div class="row ex-highlight">
    <div class="col">
        <p>You need to change everything within the 'row' to the new code.</p>
      </div>
    </div>
</section>

Usine .col-border-highlight if you need to highlight the border of a column.

<div class="col-4 offset-sm-2 col-border-highlight">

Top and Bottom spacing for button examples of code

Add the .ex-btn class to a live example of button code.

<section class="container">
<div class="row ex-btn">
  code...
</div>
</section>

Breadcrumbs

To change the color of the breadcrumb background, add a class of .breadcrumb-container to the list element.

To the change color of the links, add .breadcrumb-link to each list item.

<nav aria-label="breadcrumb">
<ol class="breadcrumb" class="breadcrumb-container">
<li  class="breadcrumb-link class="breadcrumb-item"><a href="#">Home</a></li>
<li  class="breadcrumb-item active" aria-current="page">Library</li>
</ol>
</nav>
